# Playwright Bidding Automation

A collection of Playwright scripts for automating order bidding on [EssayShark](https://essayshark.com) as a writer, along with some supporting files and demos.

## Files

### Core Scripts

- **`auto_bidder_v1.js`** — Continuous bidding bot that runs in an infinite loop. Loads the orders page, clicks the first available order, and places a bid at the minimum price. Includes dialog handling. No timeout on the bid apply step.

- **`auto_bidder_v2.js`** — Same as v1, with a 155-second timeout on the bid apply step.

- **`batch_bid_orders.js`** — One-shot script that fetches all visible order links on the orders page and bids on each one sequentially. Good for catching up on a backlog of orders.

- **`login_and_bid_prototype.js`** — Early prototype used for development and testing. Logs in with credentials, saves cookies to `login_cookies.json`, and tests the bid flow on a single hardcoded order.

### Config & Auth

- **`playwright.config.ts`** — Standard Playwright configuration (test runner settings, browser targets).
- **`package.json`** / **`package-lock.json`** — Node.js project dependencies.
- **`login_cookies.json`** — Saved session cookies for EssayShark. Generated by `login_and_bid_prototype.js` and consumed by the bidding bots.

### Supporting / Demo Files

- **`new_order_dialog_mockup.html`** — Static HTML mockup of the EssayShark "New assignment" popup (ZebraDialog). Used for testing dialog-handling logic without needing a live session.

- **`dom_edit_demo.js`** — Playground script that opens Google and changes the "I'm Feeling Lucky" button text via Playwright. Used for learning DOM manipulation.

- **`js_oop_keypress_demo.html`** — Simple OOP JavaScript example (from MDN) with a keypress listener. Used for learning purposes.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Generate session cookies by running the prototype script once (requires valid credentials):
   ```bash
   node login_and_bid_prototype.js
   ```

3. Run a bidding script:
   ```bash
   node auto_bidder_v2.js
   # or
   node batch_bid_orders.js
   ```
